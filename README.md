# HelpDeskPro

HelpDeskPro adalah aplikasi manajemen layanan IT berbasis web yang membantu organisasi mengelola siklus hidup permintaan bantuan (ticket) secara end-to-end.

## Teknologi Utama
- **Backend:** NestJS, Prisma, PostgreSQL, Redis, RabbitMQ
- **Frontend:** Next.js, TypeScript

## Dokumentasi
Silakan merujuk pada `HelpDeskPro_PRD_v3.md` untuk spesifikasi lengkap.
Untuk petunjuk setup environment, lihat `SETUP.md`.
